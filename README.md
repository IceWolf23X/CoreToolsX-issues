# CoreToolsX

CoreToolsX is a vanilla-friendly tool progression plugin for Minecraft servers.

All official information about the plugin will be available on the CoreArmorX/CoreToolsX project website:

> **Website:** https://icewolf23x.github.io/CoreArmorX-website/

The website is intended to contain the complete and up-to-date information about:

- plugin overview
- features
- installation
- configuration
- FAQ
- troubleshooting
- downloads / releases
- platform support
- Paper setup
- tool progression
- netherite tool skins
- optional resource pack addon

This GitHub repository is kept only for issue tracking.

---

## Resource Pack Addon

CoreToolsX includes an optional resource pack addon for servers using the REAL_TIER inventory model mode.

> **Download:** https://github.com/IceWolf23X/CoreToolsX-issues/releases/download/coretoolsx-resource-pack-2026.1.0/CoreToolsX-resource-pack-2026.1.0.zip

The resource pack is not required for the default plugin setup.

## Issue Tracking

Use the GitHub **Issues** section to report problems, request improvements, or track known bugs.

Before opening an issue, please check the official website first:

> https://icewolf23x.github.io/CoreArmorX-website/

When opening an issue, include as much useful information as possible.

For bug reports, include:

- CoreToolsX version
- server platform
- Paper version
- Java version
- installed permission plugin, if progression locks are enabled
- resource pack status, if REAL_TIER inventory models are enabled
- relevant config sections
- console errors or stack traces
- clear steps to reproduce the issue
- expected behavior
- actual behavior

For feature requests, include:

- what the feature should do
- why it would be useful
- any examples of the desired behavior
